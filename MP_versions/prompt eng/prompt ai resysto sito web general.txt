migliora il seguente prompt da dare all'ai per generare un sito web.

Crea il codice HTML/CSS/JS per una pagina di un sito web moderno dedicato a una webapp di cybersecurity (nome: "resysto"). 
Il design deve essere accattivante, tech, cyber, chiaro e professionale, adatto a un sito di una software company. 

Linee guida di stile:
- Layout responsive con griglie e sezioni modulari.
- Dark mode di default con palette elegante: sfondi scuri, accenti blu/verde/viola in stile cyber-tech.
- Font moderni e geometrici (es. Inter, Space Grotesk, JetBrains Mono per dettagli tecnici).
- Animazioni fluide e micro-interazioni (hover glow, transizioni morbide, effetti neon discreti).
- Iconografia minimal (emoji o icone SVG).
- Struttura leggibile e chiara, pensata sia per decision maker (CISO, CIO, CEO) che per utenti tecnici.

Elementi riutilizzabili in qualsiasi pagina:
- **Hero section** con kicker, titolo, sottotitolo e bottoni CTA (primario e secondario).
- **Content grid / card layout** per mostrare moduli, features, servizi o pricing plans.
- **Status badges** con codice colore coerente (verde=complete, giallo=beta, viola=in dev, blu=planned, arancio=experimental).
- **Two-column layout** per alternare testo descrittivo e visual/card.
- **Key concepts / highlights section** con icone + testo breve.
- **Footer minimale** con brand, link utili e note sul design.

Requisiti extra:
- Accessibilità: contrasti alti, font leggibili, conforme WCAG 2.1 AA.
- Struttura modulare e facilmente adattabile a più pagine (Features, About, Docs, Pricing).
- Look cyberpunk elegante: dettagli neon glow molto sottili, senza eccessi.

Output richiesto:
- Codice HTML semanticamente corretto con sezioni chiare.
- CSS integrato (Tailwind o custom con variabili).
- Struttura pronta da adattare in framework statici (Hugo, Astro, Next.js).
